# Permiso Ver los detalles de las ocupaciones en el calendario

El nuevo permiso **Ver los detalles de las ocupaciones en el calendario** diferencia la capacidad del usuario de ver el formulario de ocupaciones y de guardarlo.

Ahora podremos asignar permisos a usuarios de festia para que puedan ver los detalles de las ocupaciones pero que no puedan modificarlas.

Ahora podremos asignar a un usuario el perfil Artista, y por defecto tendrá estos permisos:
> **NOTA:** Los datos económicos solo son visibles para el perfil **Agencia** por defecto.

![artista-permisos-ver-detalle-ocupacion.png](/img/release-notes/artista-permisos-ver-detalle-ocupacion.png)


A diferencia del perfil **Agencia** que por defecto puede modificar ocupaciones y ver datos económicos con los permisos
**Realizar ocupaciones del artista** y **Ver datos económicos de las ocupaciones del artista**.

Por ver la diferencia, cuando asignamos a un usuario el perfil **Agencia** tienen todos estos permisos:

![agencia-permisos.png](/img/release-notes/agencia-permisos.png)

Como siempre, cada uno de estos permisos es configurable dentro de vuestra agencia, pudiendo añadir y eliminar permisos a cada usuario individualmente.
